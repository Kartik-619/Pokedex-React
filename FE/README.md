<h1>🎮 Pokémon Explorer App</h1>
<br>
A simple React application that lets you search for Pokémon, view their details, play their cry sounds, and explore basic information like types and moves.
<br>
<br><h3>🔍 Features</h3>
<br>✅ Search any Pokémon by name
<br>🖼️ View the Pokémon's official sprite image
🔊 Play the Pokémon's cry sound<br>
<br>ℹ️ Toggle to view additional info: types and sample moves
<br>📱 Responsive design for mobile and desktop
<br>🛠 Technologies Used
<br>React – For building the user interface
<br>Tailwind CSS – For styling
<br>PokeAPI – To fetch Pokémon data
<br>PokemonCries.com – For Pokémon cry audio files
<br><br>
<h3>📦 Installation & Setup</h3>
Clone the repository<br>
bash<br>


<br>git clone https://github.com/your-username/pokemon-explorer.git 
<br>cd pokemon-explorer
<br>Install dependencies
<br>bash



<br>npm install
<br>Run the app
<br>bash



<br>npm start
<br>The app will start running at http://localhost:3000.
<br><h3>💡 How to Use</h3>
<br>Type a Pokémon name in the input field (e.g., "Bulbasaur", "Charizard")
<br>Click Search or press Enter
<br>You'll see the Pokémon's image and name appear above
<br>Click the 🔊 button to play the Pokémon’s cry
<br>Click the Info button to toggle detailed info like type(s) and move list
